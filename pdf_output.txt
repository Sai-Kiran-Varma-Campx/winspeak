DESIGN CHANGE BRIEF 
WinSpeak School POC — UI & UX Edits 
This document specifies all proposed changes to the WinSpeak school POC for the original WinSpeak POC. 
Each row defines one element: what it currently shows, what it should show, and the full specification for 
implementation.

  Skill 
Breakdown 
↳ Clarity — 
Assessment 
Scope 
Pronunciation and 
word clarity only 
Expanded to include 
voice modulation 
and voice throw 
The Clarity skill card must assess four 
dimensions: (1) pronunciation — how 
clearly individual words are spoken, (2) 
word clarity — whether the listener can 
understand what is being said, (3) voice 
modulation — variation in pitch and tone 
to convey meaning and maintain listener 
engagement, (4) voice throw — 
projection and volume appropriate to the 
speaking context. 
All four dimensions feed into the single 
Clarity star score for this card.

Default and only mode is light. 
Dark mode removed entirely. 
Backgrounds: white or soft pastels — sky 
blue, warm yellow, mint green. 
Typography: rounded, friendly typefaces

Icons: simple emoji-style or cartoon icons

Score Display 
Numeric score 
shown (e.g. 
88/100) 
Star rating only — 
no numbers 
No numeric score shown to students or 
parents at any point. 
Conversion formula: Stars = 
CEILING(score ÷ 20, 0.5), giving a 0–5 
value rounded up to the nearest 0.5. 
Examples: 82 → 4.5 stars; 60 → 3.0 
stars; 45 → 2.5 stars; 0 → 0 stars. 
UI must support both full-star and half-
star rendering.

portal

All references to XP points must be 
removed from every page across the 
portal.

Each card displays: (1) a simple icon, (2)

TEACHER PORTAL PAGES 
Teacher Home 
Not present 
New page — 
primary landing 
screen after login 
This is the primary landing page a 
teacher sees after logging in. 
Displays the teacher's class roster with 
each student's name and most recent 
challenge date. 
Two primary actions are available from 
this page: Add Student and Administer 
Challenge. 
  ↳ Add Student 
Not present 
"Add Student" 
action and form 
A button on Teacher Home opens an 
Add Student form. 
Form fields: full name, student ID, grade, 
section, parent email (optional). 
On submit, the student is added to the 
teacher's class roster immediately. 
  ↳ Administer 
Challenge 
Not present 
"Administer 
Challenge" action 
A button on Teacher Home initiates the 
Challenge Administration Flow. 
See Challenge Administration Flow 
below for the full step-by-step 
specification. 
Challenge 
Administration 
Flow 
Not present 
New 3-step flow — 
triggered from 
Teacher Home 
A three-step sequential flow initiated by 
clicking Administer Challenge on the 
Teacher Home page. 
The teacher cannot proceed to the next 
step without completing the current one. 
  ↳ Step 1 — 
Category 
Not present 
Category selection 
screen 
Teacher selects one category from 8 
options: Circletime, Building Talks, 
TEDlets, Interview Discussion, Voice for 
Change, Podcast Playground, Student 
Council Speeches, Policy Proposals. 
Only one category can be selected per 
session. 
  ↳ Step 2 — 
Grade & 
Question 
Not present 
Grade and question 
selection screen 
Teacher selects a grade (1–4) and a 
specific question from within the selected 
category and grade. 
Questions are filtered by the category 
and grade selected in Step 1. 
  ↳ Step 3 — 
Administer 
Not present 
Per-student 
administration 
screen 
Displays the full class roster for the 
session. 

--- PAGE 4 ---
Teacher marks each student individually 
— Start when the student begins 
speaking, Done when they finish. 
On Done, the system generates and 
saves an individual report for that 
student. 
Students are assessed one by one; the 
teacher moves through the roster 
sequentially. 
Teacher 
Dashboard 
Not present 
New page — class 
overview 
Displays a class-level summary for all 
students in the teacher's section. 
Columns: student name, number of 
challenges completed, average star 
rating, top strength, area to improve. 
No XP points, streak indicators, or 
gamification metrics of any kind.
